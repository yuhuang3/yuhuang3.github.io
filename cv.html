Yu, M.D., M.S. in computer science, was a principal Software Engineer/Business Intelligence Analyst at Argonne National Lab with demonstrated tracking records in data analytics, data & software engineering, and machine learning in biomedical field, industry, university and national lab.


  1. 1. Machine learning:
    • (a) Collaborated with colleages in CELS Argonne in studying electronic health records (EHR) using deep learning and NLP methods.
    • (b) Authored and submitted a research proposal on machine learning in synthetic biology.
    • (c) Authored articles on explainable machine learning and deep learning in Towards Data Science

  2. 2. Data analytics:
    • (a) Extensive working experiences in SAP Business Intelligence (BusinessObjects) software to design Universes and Objects for data Analytics and data visualization.
    • (b) Extensive working experiences in data science stack (Python, Numpy, Pandas, Matplotlib, Jupyter Notebook), SQL and PLSQL, Oracle database, and data warehouse
    • (c) Responsible for generating DOE APS scientific user facility annual report and other decision making reports for ANL Lab management.

  3. 3. Software development
    • (a) Played a leading role in monitoring and selecting modern technologies for software development.
    • (b) Applied modern technologies, especially open source tools to software development at Photon Science scientific user facility in Argonne National Laboratory: Docker micro-services, React Mobile App, JavaScript framework (angular, react.js/react native.js, node.js, expo, etc.), Python, Flask, Nginx, Gunicorn, etc.
    • (c) Responsible for whole software development life cycle (with agile process): project planning, software requirements gathering, analysis, design, development (coding), testing, deployment, and maintenance.


Yu Received multiple Argonne national lab Pacesetter and Impact Argonne Awards in the past seventeen years for the outstanding work in data analytics, business intelligence, data engineering, and software development.


Yu was an active member of Argonne Women in Science and Technology (Steering Committee Member since 2014).


Yu authored multiple articles on machine learning and deep learning in Towards Data Science. More information is available in LinkedIn.
